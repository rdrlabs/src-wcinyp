# Introduction to WCINYP

Welcome to the Weill Cornell Imaging at NewYork-Presbyterian (WCINYP) administrative system. This comprehensive platform streamlines medical imaging operations across all WCINYP locations.

## What is WCINYP?

WCINYP is a unified administrative platform designed to manage:

- Patient documents and forms
- Provider information and directories
- Automated form generation
- Comprehensive contact management
- Knowledge base and documentation

## Key Benefits

### 1. Centralized Document Management

All patient forms, questionnaires, and administrative documents are organized in one secure location:

- **ABN Forms**: Location-specific Advanced Beneficiary Notices
- **Patient Questionnaires**: Modality-specific intake forms
- **Administrative Forms**: Waivers, authorizations, and consent forms
- **Invoice Templates**: Standardized billing documentation

### 2. Efficient Provider Management

Quickly access and manage provider information:

- Contact details and specialties
- Department assignments
- Location information
- Direct communication links

### 3. Automated Form Generation

Save time with intelligent form automation:

- Pre-populated patient information
- Location-specific form selection
- Multi-language support
- Digital signature capabilities

### 4. Comprehensive Directory

Master contact database includes:

- Healthcare providers
- Insurance companies
- Laboratory partners
- Facility contacts
- Vendor information

## Getting Started

### Step 1: Access the System

Navigate to the WCINYP dashboard where you'll see all available modules:

- **Document Hub**: Access forms and documents
- **Provider Directory**: Search provider information
- **Form Generator**: Create automated forms
- **Directory**: Browse all contacts
- **Knowledge Base**: Access documentation

### Step 2: Choose Your Module

Each module is designed for specific tasks:

| Module | Primary Use |
|--------|------------|
| Documents | Download and manage forms |
| Providers | Find provider contact info |
| Form Generator | Create patient forms |
| Directory | Access all contacts |
| Knowledge Base | Learn system features |

### Step 3: Use Search Features

Every module includes powerful search capabilities:

- Filter by category
- Search by keyword
- Sort by date or name
- Export results

## Navigation Tips

### Global Features

- **Dark Mode**: Toggle in the top navigation bar
- **Search**: Available in every module
- **Breadcrumbs**: Track your location in the system

### Keyboard Shortcuts

- `Ctrl/Cmd + K`: Global search
- `Ctrl/Cmd + /`: Toggle help
- `Esc`: Close modals

## System Architecture

WCINYP is built with modern technologies:

- **React Router v7**: Fast, modern routing
- **shadcn/ui**: Accessible UI components
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Responsive design
