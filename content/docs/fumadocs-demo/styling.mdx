## Custom Themes

### Method 1: CSS Variables

Override the default CSS variables in your global CSS:

```css title="app/globals.css"
:root {
  /* Override primary color */
  --primary: 220 90% 56%;
  --primary-foreground: 0 0% 100%;
  
  /* Custom radius */
  --radius: 0.375rem;
  
  /* Custom font */
  --font-sans: "Inter", sans-serif;
}
```

### Method 2: Tailwind Config

Extend your Tailwind configuration:

```javascript title="tailwind.config.js"
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: 'hsl(var(--primary))',
          foreground: 'hsl(var(--primary-foreground))',
        },
      },
      fontFamily: {
        sans: ['Inter', 'sans-serif'],
      },
    },
  },
}
```

### Method 3: Custom Components

Override default components with your own:

```tsx title="components/custom-callout.tsx"
export function Callout({ children, type = 'default', ...props }) {
  return (
    <div 
      className={`my-custom-callout ${type}`} 
      {...props}
    >
      {children}
    </div>
  );
}
```

  3. Use CSS variables for consistency
  4. Keep animations subtle and purposeful
  5. Test on various screen sizes
