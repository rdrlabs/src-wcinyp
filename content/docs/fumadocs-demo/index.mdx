# Welcome to Fumadocs

Fumadocs is a documentation framework for Next.js, designed to help developers build beautiful documentation sites with MDX, full-text search, and internationalization support.

<Callout type="tip">
  This is a default Fumadocs demonstration showcasing all the out-of-the-box features. Explore the pages to see what Fumadocs can do!
</Callout>

## What is Fumadocs?

Fumadocs provides everything you need to create modern documentation:

## Quick Start

<Steps>
  <Step>
    ### Create a Next.js App
    
    Start with a fresh Next.js application:
    ```bash
    npx create-next-app@latest my-docs --typescript --tailwind --app
    cd my-docs
    ```
  </Step>
  
  <Step>
    ### Install Fumadocs
    
    Add Fumadocs and its dependencies:
    ```bash
    npm install fumadocs-ui fumadocs-core fumadocs-mdx
    ```
  </Step>
  
  <Step>
    ### Configure Fumadocs
    
    Set up your documentation structure and configuration files
  </Step>
  
  <Step>
    ### Start Writing
    
    Create your first MDX page and see it live!
  </Step>
</Steps>

## Project Structure

A typical Fumadocs project looks like this:

<Files>
  <Folder name="app" defaultOpen>
    <Folder name="docs">
      <File name="[[...slug]]/page.tsx" />
      <File name="layout.tsx" />
    </Folder>
    <File name="layout.tsx" />
    <File name="page.tsx" />
  </Folder>
  <Folder name="content" defaultOpen>
    <Folder name="docs" defaultOpen>
      <File name="index.mdx" />
      <File name="getting-started.mdx" />
      <Folder name="guides">
        <File name="installation.mdx" />
        <File name="configuration.mdx" />
      </Folder>
      <File name="meta.json" />
    </Folder>
  </Folder>
  <File name="source.config.ts" />
  <File name="next.config.mjs" />
</Files>
