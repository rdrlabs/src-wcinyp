# Quick Start Guide

## Welcome to WCINYP

This guide will help you get started with the Weill Cornell Imaging at NewYork-Presbyterian system.

## First Steps

### 1. Access the System

Navigate to the WCINYP portal and log in with your credentials.

### 2. Explore Key Features

- **Documents Hub**: Browse and download medical forms
- **Provider Directory**: Find contact information for medical staff
- **Form Builder**: Create custom forms for your needs
- **Contact Directory**: Access comprehensive contact information

### 3. Common Tasks

#### Finding Documents
1. Go to the [Documents](/documents) section
2. Use the search bar or browse by category
3. Click download to get the form you need

#### Searching for Providers
1. Visit the [Providers](/providers) page
2. Search by name, department, or location
3. View detailed contact information

#### Creating a Form
1. Access the [Form Builder](/forms)
2. Select a template or start from scratch
3. Add fields and customize as needed
4. Save or export your form

## Need Help?

If you need assistance, check our [Contact Directory](/directory) for support contacts.
