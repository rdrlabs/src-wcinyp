# System Requirements

Ensure your system meets these requirements for optimal WCINYP performance.

## Supported Browsers

WCINYP works best with modern web browsers. We recommend using the latest versions of:

- **Google Chrome** (version 90+)
- **Mozilla Firefox** (version 88+)
- **Microsoft Edge** (version 90+)
- **Safari** (version 14+)

## Operating Systems

WCINYP is web-based and works on:

- **Windows**: Windows 10 or later
- **macOS**: macOS 10.15 (Catalina) or later
- **Linux**: Ubuntu 20.04 LTS or equivalent
- **iOS**: iOS 14 or later (for mobile access)
- **Android**: Android 10 or later (for mobile access)

## Hardware Requirements

### Minimum Requirements
- **Processor**: 1.6 GHz or faster
- **RAM**: 4 GB
- **Storage**: 500 MB free space for downloads
- **Display**: 1366x768 resolution
- **Internet**: Broadband connection (5 Mbps)

### Recommended Requirements
- **Processor**: 2.0 GHz or faster (multi-core)
- **RAM**: 8 GB or more
- **Storage**: 2 GB free space for downloads
- **Display**: 1920x1080 resolution or higher
- **Internet**: High-speed broadband (25 Mbps or faster)

## Network Requirements

### Firewall Settings
Ensure your firewall allows access to:
- HTTPS (port 443)
- WebSocket connections
- WCINYP domain and subdomains

### Proxy Configuration
If using a corporate proxy:
- Configure proxy exceptions for WCINYP domains
- Ensure proxy supports WebSocket connections
- Contact IT for proper configuration

## Additional Software

### PDF Viewer
- Adobe Acrobat Reader DC or
- Built-in browser PDF viewer

### Optional Tools
- Microsoft Office or compatible suite (for exports)
- Email client for notifications
- Calendar application for scheduling

## Mobile Access

WCINYP offers limited mobile functionality:
- View documents
- Search directories
- Basic form submission
- Contact information access

For full functionality, use a desktop or laptop computer.

## Troubleshooting

### Performance Issues
1. Clear browser cache and cookies
2. Disable unnecessary browser extensions
3. Close unused tabs and applications
4. Check internet connection speed

### Display Problems
1. Update graphics drivers
2. Adjust browser zoom (Ctrl/Cmd + 0 to reset)
3. Check display resolution settings
4. Try a different browser

### Connection Errors
1. Verify internet connectivity
2. Check firewall settings
3. Disable VPN temporarily
4. Contact IT support
