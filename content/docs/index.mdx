Welcome to the Weill Cornell Imaging at NewYork-Presbyterian (WCINYP) documentation. This knowledge base contains everything you need to know about using the WCINYP system effectively.

## Key Features

### Document Management
Manage patient forms, questionnaires, and administrative documents efficiently.

### Provider Directory
Access comprehensive information about medical staff, departments, and specialties.

### Form Generator
Automate document creation and streamline the self-pay form process.

### Master Directory
Centralized database for all contacts including providers, insurance, facilities, and vendors.
